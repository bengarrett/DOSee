# Security Policy

DOSee uses GitHub Dependabot for automated security updates.

## Supported Versions

Only the latest version receives security updates.

## Reporting Vulnerabilities

Report security issues via [Bug Report](https://github.com/bengarrett/DOSee/issues/new?assignees=bengarrett&labels=vulnerability&template=bug_report.md&title=%5BBUG%5D) with the vulnerability label.

Do not publicly disclose vulnerabilities before they are fixed.
