# Contributing to DOSee

**Quick Start:**
```bash
git clone https://github.com/bengarrett/DOSee.git
cd DOSee
pnpm install
pnpm dev
```

**Contribution Guidelines:**
- Fork repository, create feature branch
- Follow existing code style and patterns
- Test thoroughly in current browsers (Chrome, Firefox, Edge, Safari)
- Must support WebAssembly and modern JavaScript features
- Submit pull requests with clear descriptions
- **Note:** Contributions may not be accepted if they don't align with project goals

**Reporting Issues:**
- Use GitHub Issues template
- Include reproduction steps and browser details
- Be specific about environment and expected behavior

All contributions welcome, but final acceptance is at maintainer discretion.
